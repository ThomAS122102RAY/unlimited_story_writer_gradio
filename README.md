# 📝 AI 深度寫作助手 v2.2 - 安裝與使用指南

你好！這是利用你的顯卡進行本地運算的 AI 寫作工具。因為是私有架設，完全免費且隱私安全。請依照以下步驟進行安裝：

---

## ⚠️ 電腦建議規格

* **系統**：Windows 10 或 11
* **顯卡**：建議使用 NVIDIA 顯卡（VRAM 8GB 以上體驗較佳）。
* **硬碟**：請預留約 20GB 空間（用於下載 AI 模型）。

---

## 第一步：安裝環境 (只需做一次)

### 1. 安裝 Python
* 前往官網下載：[Python 3.10 下載連結](https://www.python.org/downloads/)
* **重要：** 安裝時，務必勾選最下方的 **"Add Python to PATH"** (加入環境變數)，否則程式會無法執行。

### 2. 安裝 AI 引擎 (Ollama)
* 前往官網下載並安裝：[Ollama 官網](https://ollama.com/)
* 安裝完畢後，請確認螢幕右下角的工作列有出現一個「羊駝」的小圖示。

### 3. 下載 AI 模型
* 按下鍵盤 `Win + R`，輸入 `cmd` 後按 Enter 打開黑底視窗。
* 複製貼上以下指令並按 Enter（下載約 15GB，請耐心等待）：
```bash
ollama run gemma2:27b
```
*(備註：如果你的顯卡記憶體小於 10GB，覺得跑不動，請改輸入 `ollama run gemma2:9b`)*
* 當看到對話游標出現，代表成功。直接關閉視窗即可。

---

## 第二步：設定寫作助手 (只需做一次)

1. 將壓縮包解壓縮到桌面或你喜歡的地方。
2. 進入解壓後的資料夾。
3. 在資料夾的**上方網址列**點一下，輸入 `cmd` 並按下 Enter。
4. 在跳出的黑底視窗中，輸入以下指令來安裝必要套件：
```bash
pip install -r requirements.txt
```
5. 等待跑完，出現 `Successfully installed...` 字樣後，關閉視窗。

---

## 第三步：開始寫作

1. 雙擊資料夾中的 **`啟動.bat`**。
2. 會跳出一個黑色視窗，**請不要關閉它**。
3. 瀏覽器會自動開啟寫作介面（如果沒開，請手動輸入網址 `http://127.0.0.1:7860`）。
4. 開始享受你的創作！

---

## ❓ 常見問題

* **Q: 打不開，顯示 Connection refused？**
  * A: 請確認右下角的 Ollama (羊駝圖示) 有在執行。

* **Q: 黑色視窗一閃就退？**
  * A: 通常是 Python 沒裝好，或是沒有執行第二步的安裝套件指令。

* **Q: 生成速度很慢？**
  * A: 這是本地端運算，取決於你的顯卡效能。可以在右側控制台調整「生成長度」短一點，或是改用較小的模型 (gemma2:9b)。
